# SQL



## Commands

```sh
\q # To quit the psql:
\l # List all databases 
\dv # List all views:
\dt # Lists all tables in a current database. 
\du # List all users
```

## Database

```sh
CREATE DATABASE [IF NOT EXISTS] db_name;
DROP DATABASE [IF EXISTS] db_name;

```

## Tabls

```sh
SELECT * FROM table_name;
```
